# Intent

Make system progress an explicit, durable, **deterministically replayable** sequence of validated state transitions — never application logic scattered across services. The machine is durable and crash-recoverable: it can never enter an undefined state, and every committed decision survives a crash. Crucially, the durable log records the accepted `Decision` data, so a run can be **re-executed to byte-identical state**, not merely recovered to its last state — recovery is a special case of replay. Distribution is single-node-first, with bounded multi-node coordination over the shared file system; it adds no cluster-scale replication, consensus service, or broker. This is the deterministic backbone the Prime Directive demands — durable and replayable, never claimed "unbreakable."
